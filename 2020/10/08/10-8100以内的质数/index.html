10.8循环-100以内的质数

思路
质数：只能被1和它本身整除的自然数

利用循环,如果这个数可以与其他数取余为0，说明不是质数。

不是质数的数都可以拆成两个数相乘,也就是成对出现的，所有循环的条件到这个数的根号就行了

public class PrimeNumber {
    public static void main(String[] args) {

        for(int i= 3 ;i<=100;i++){
            boolean isFlag = true;
            //优化二：对本身是质数的自然数是有效的
            for (int j =2;j <= Math.sqrt(i);j++){
                if(i%j==0){
                    //可以除尽，说明不是质数退出当前循环
                    isFlag = false;
                    break;//优化1：只对本身非质数的自然数是有效的
                }
            }
            if(isFlag){
                System.out.println(i + "是质数");
            }
        }
    }
}

优化
利用标签可以有更加简单的写法

public class PrimeNumber {
    public static void main(String[] args) {

       label:for(int i= 3 ;i<=100;i++){
            for (int j =2;j <= Math.sqrt(i);j++){
                if(i%j==0){
                    continue label;
                }
            }
                System.out.println(i + "是质数");
        }
    }
}
