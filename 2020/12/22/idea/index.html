idea使用笔记

Module

project 工程下可以新建 module ,每个module下都有一个src目录。

顶级 项目project –> 多个模板module(基本根据功能划分)

删除module

方法1.右键remove module
方法2.右键open module settings -> 点减号

并没有删掉，模块变成普通的文件夹，普通的文件目录邮件delete就可以直接删除了。

快捷键

复制

ctrl + D 复制该行并输出在该行下一行

换行

ctrl + shift + enter 任何位置的换行

快捷开发

alt+shift+S 选中代码块包裹，try-catch等等
alt+shift+delete 删除代码包裹块
alt + enter 错误解决办法
ctrl + p 光标停在new类的括号中，显示可选形参
alt + ins 快速生成代码

模板

位置1：Editor - General - Postfix completion
此位置不可以修改

位置2: Editor - Live Templates - other/output
此位置可以自己修改与添加

按Tab键切换光标位置

sout

x.sout + Tab = System.out.println(x);

soutp + Tab = System.out.println(“打印当前方法的形参”);

main

main + Tab

public static void main(String[] args){
$END$
}

fori + Tab

for (int i = 0; i < ; i++) {
}
