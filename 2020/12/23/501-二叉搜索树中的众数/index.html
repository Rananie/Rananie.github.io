501.二叉搜索树中的众数

题目

给定一个有相同值的二叉搜索树（BST），找出 BST 中的所有众数（出现频率最高的元素）。

假定 BST 有如下定义：

结点左子树中所含结点的值小于等于当前结点的值
结点右子树中所含结点的值大于等于当前结点的值
左子树和右子树都是二叉搜索树
例如：
给定 BST [1,null,2,2],

1
\
2
/
2
返回[2].

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/find-mode-in-binary-search-tree

思路

首先是一个二叉搜索树，那么根据二叉搜索树的特点，可以得知中序遍历是单独非增的，那么套上中序遍历的模板。

//根据中序遍历单调不减的特点，相同的元素个数应该会出现在左右两边
TreeNode pre = null; //记录前一个结点
int count = 1; //统计评论
int maxCount = 1 ; //最大频率
public void inorder(TreeNode root){
    if(root == null) return ;
    inorder(root.left);
    //处理节点
    inorder(root.right);
    }

先处理频率,累计当前数出现的频率，如果当前和pre不相等则重置

 if(pre != null &&  root.val == pre.val){
 count ++ ;
 }else{
 //如果不相等,重置
 count = 1 ;
 }

因为需要返回的是最大频率的，这里有个技巧就是当count == maxCount 时先把该数放进去，该数不一定是最大频率的，然后当 count>maxCount 时，更新maxCount，并清空数组，重新放进去

if(pre != null &&  root.val == pre.val){
    count ++ ;
}else{
    //如果不相等,重置
    count = 1 ;
}
if(count == maxCount){
    answer.add(root.val);
    //放入数组中
}else if(count > maxCount){
    //说明不是出现次数最多数，清空数组，更新maxCount
    answer.clear();
    answer.add(root.val);
    maxCount = count;
}
//更新当前值
pre = root;

代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    //根据中序遍历单调不减的特点，相同的元素个数应该会出现在左右两边
    TreeNode pre = null; //记录前一个结点
    int count = 1; //统计评论
    int maxCount = 1 ; //最大频率
    List<Integer> answer = new ArrayList<>();//答案数组
    public int[] findMode(TreeNode root) {
        inorder(root);
        //JAVA数组的一些特点
         int[] mode = new int[answer.size()];
         for (int i = 0; i < answer.size(); ++i) {
            mode[i] = answer.get(i);
        }
        return mode;
    }
    public void inorder(TreeNode root){
        if(root == null) return ;
        inorder(root.left);
        //处理节点
        if(pre != null &&  root.val == pre.val){
            count ++ ;
        }else{
            //如果不相等,重置
            count = 1 ;
        }
        if(count == maxCount){
            answer.add(root.val);
            //放入数组中
        }else if(count > maxCount){
            //说明不是出现次数最多数，清空数组，更新maxCount
            answer.clear();
            answer.add(root.val);
            maxCount = count;
        }
        //更新当前值
        pre = root;
        inorder(root.right);
    }
}
