java.util.Date类
1.两个构造器的使用
构造器1：new Date() 创建一个对应当前时间的Date对象。
构造器2：new Date(long date):形参为毫秒数，创建指定毫秒数的Date对象
2.两个方法的使用
toString():显示当前的年、月、日、时、分、秒
getTime():获取当前date对象对应的时间戳(1970年1月1日0时0分0秒之间的毫秒数)

java.sql.Date类对应着数据库中的日期类型的变量
1.如何实例化 new java.sql.Date(long date)
2.java.sql.Date对象 –> java.util.Date对象 向上转型直接赋值
3.java.util.Date对象 –> java.sql.Date对象 

Data date = new Date();
java.sql.Date date1 = new java.sql.Date(date.getTime())

不管向上转还是向下转，前提是实例包含这个结构，new的父类，不能直接强转成子类。

java.text.SimpleDateFormat类

Data类的API不易于国际化,大部分被废弃了。java.text.SimpleDateFormat类是一个不与语言环境有关的方式来格式胡和解析日期的具体类。

SimpleDateFormat的使用:对日期Date类的格式化和解析

两个操作：
1.格式化：日期 –> 字符串
2.解析： 字符串 –> 日期

SimpleDateFormat的实例化

//默认构造器用的是默认的格式
SimpleDateFormat sdf = new SimpleDateFormat();
//格式化
Date date = new Date()
String format = sdf.form(date);
//解析
Date date = sdf.parse(format);

//指定格式的字符串 年yyyy-MM月份-dd天 hh:mm:ss
SimpleDateFormat sdf = new SimpleDateFormat("指定格式的字符串");

/*
练习1：字符串“2020-09-08”转化为java.sql.Date
*/

public void testExer(){
    String birth = "2020-09-08";
    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
    Date date = sdf.parse(birth); //先转化成ava.util.Date
    java.sql.Date date1 = new java.sql.Date(date.getTime())
}

JDK8中新日期时间API

JDK8之前面临的问题：
1.可变性:像日期和时间这样的类应该是不可变的。
2.偏移性：Date中的年份是从1900开始的，而月份都是从0开始。(year + 1900,month + 1,date)
3.格式化：格式化只对Date有用，Calendar则不行。
4.并不是线程安全的。

java.time

LocalDate,LocalTime,LocalDateTime

1.实例化

静态方法
now()方法:获取当前的日期、时间、日期+时间
of()方法:设置指定的时间,没有偏移量

2.操作方法，实例方法
getXxx() 获取相关属性
withXxx() 设置相关属性，有返回值，返回值是修改后的日期，原来的不改变，不可变性。
plusXxx()/minusXxx() 对相关属性进行增加删除操作

//now()获取当前的日期、时间、日期+时间
//2020-09-09
LocalDate localDate = LocalDate.now()
LocalTime localTime = LocalTime.now()
LocalDateTime localDateTime = LocalDateTime.now()

//of（）:设置指定的时间,没有偏移量
LocalDateTime localDateTime = LocalDateTime.of(year,month,dayOfMonth,hour,minute,second)

Instant

时间线上的一个瞬时点，可以用来记录应用程序中的事件时间戳。是一个数以1970年开始，以毫秒为单位。
类似 java.util.Date 类

1.实例化

now():获取本初子午线对应的标准时间
ofEpochMilli(long毫秒数):通过毫米数实例化instant ->Date(long millis)

//有时差，按照本初子午线的时间
Instant instant = Instant.now();
//实例方法，添加时间的偏移量
OffsetDateTime atOffsetDateTime = instant.atOffset(ZoneOffset.ofHours(8)); //设置时区
Instant instant = Instant.ofEpochMilli(milli);

2.操作

instant.toEpochMilli():获取对应的毫秒数 -> Date类的getTime()

//获取自1970年1月1日0时0分0秒(UTC)开始的毫秒数
long milli = instant.toEpochMilli();

java.time.format.DateTimeFormatter类

DateTimeFormatter：格式化或解析日期、时间。类似与SimpleDateFormat

1.实例化
预定义的标准格式：如ISO_LOCAL_DATA_TIME
本地化相关的格式: 如ofLocalzedDateTime()。参数FormatStyle.Long/FormatStyle.MEDIUM/FormatStyle.SHORT 适用于LocalDateTime
自定义的格式(常用):ofPattern(“yyyy-MM-dd hh:mm:ss E”)

//实例化
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd hh:mm:ss E");
//格式化 日期 -> 字符串
String str = formatter.format(LocalDateTime/LocalDate/LocalTime) //2019-02-18 03:52:09
//解析 字符串 -> 日期
formatter.parse(str);
