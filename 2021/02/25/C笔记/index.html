前导

产生一个变量就分配空间

值传递和地址传递

默认值传递的类型：基本数据类型(整型类型、小数类型、字符类型char)，结构体，共用体

默认传递地址的类型：指针、数组

指针

& :获取变量的地址,%p输出一个变量的地址，如printf("地址=%p",&num)
* :
1.指针类型，指针变量存的是一个地址，这个地址指向的空间存的才是值。
2.如 int ptr = &num。ptr是指向int类型的指针变量，即ptr是`int类型，是指向一个int类型的变量。 3.取值符，获取指针指向的变量的值。如* ptr`在内存中指向的是num，获取到了num值

总结：
ptr –> num的地址
num、*ptr –> num的值

注意点：1.指针加的是一个单位,如下面代码加的是4个字节(一个int)

int var[] = {10,100,200}; //int数组
int i,*ptr;
ptr = var;
ptr++; //ptr存放的var的首地址值+4字节(int)

指针数组

指针数组的定义:数据类型 指针数组名[大小];
如:int ptr[3]
1.ptr声明为一个指针数组
2.由三个整数指针组成。ptr中的每个元素，都是一个指向int值的指针

注意点:2.字符串的名称就代表首地址,可以理解为数组中存放的是字符串的变量名字。例如char *pStr="abc",输出用的pStr

char *books[] ={
    "三国演义",
    "西游记",
    "红楼梦",
    "水浒传"
};
int i ,len=4;
for(i=0;i<len;i++){
    //char *pStr="abc",输出用pStr
    printf("books[%d]指向字符串是=%s",i.books[i]);
}
