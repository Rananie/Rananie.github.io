前导

产生一个变量就分配空间

值传递和地址传递

默认值传递的类型：基本数据类型(整型类型、小数类型、字符类型char)，结构体，共用体

默认传递地址的类型：指针、数组

字符串

字符串也就是字符数组，在C语言中没有string类型,字符占1个字节。

注意：
1.字符串使用null字符(‘\n’)终止一维字符数组。char str[7]={a,b,c},在内存中为a、b、c、\n。\n表示字符串结束。
2.null要占用数组的一个空间，数组有多余的空间(需要明确指定数组的长度)，则会自动添加。
3.char str[] = {a,b,c}不会自动添加。char grrrting[] = “hello”会默认添加null字符。

相关写法

char str1[] = {"I am happy"}; //默认后面加'\0'
char str2[] = "I am happy"; //省略{}号，默认后面加'\n'
char str3[] ={'I','','a','m','','h','a','p','p','y'}; //字符数组后面不会加'\0',可能有乱码 
char str4[5]= {'C','h','i','n','a'}; //字符数组后面不会加'\0'，可能有乱码
char * pStr = "hello";

字符数组与字符串
字符数组只能对各个元素赋值，不可以一起赋值。

char str[14]; //str实际是一个常量
str="hello tom"; //错误的,因为相当于对常量赋值。
str[0] = "i"; //正确的

指针

& :获取变量的地址,%p输出一个变量的地址，如printf("地址=%p",&num)
* :
1.指针类型，指针变量存的是一个地址，这个地址指向的空间存的才是值。
2.如 int ptr = &num。ptr是指向int类型的指针变量，即ptr是`int类型，是指向一个int类型的变量。 3.取值符，获取指针指向的变量的值。如* ptr`在内存中指向的是num，获取到了num值

总结：
ptr –> num的地址
num、*ptr –> num的值

注意点：1.指针加的是一个单位,如下面代码加的是4个字节(一个int)

int var[] = {10,100,200}; //int数组
int i,*ptr;
ptr = var;
ptr++; //ptr = ptr存放的var的首地址值+4字节(int)，ptr存放的地址会改变
//ptr[1] = ptr + 1个单位 ptr存放的地址不会改变

当接受值为指针时，可以传数组因为传递的是数组的首地址

指针的注意事项和细节
1.指针变量存放的是地址，从这个角度看指针的本质就是地址
2.变量声明的时候，如果没有确切的地址赋值，为指针变量赋值NULL
3.赋为NULL值的指针被称为空指针，NULL指针是一个定义在标准库stdio.h中值为零的常量#define NULL 0

指针函数与函数指针

指针函数

type * functionName(param list)

允许函数的返回值是一个指针(地址)

static修饰的变量会在静态数据区，不会随函数消亡而消亡。

指针函数注意事项和细节
1.函数运行结束后会销毁在它内部定义的所有局部数据，包括局部变量、局部数组和形式参数，函数返回的指针不能指向这些数据。
2.函数运行结束后会销毁所有的局部数据，销毁并不是将局部数据所占用的内存全部清零，而是程序放弃对它的使用权限，后面的代码可以使用这块内存。销毁之后可能后面代码没有使用该内存，所以还能返回，

函数指针

函数指针是指向函数的指针

一个函数总是占用一段连续的内存区域,函数名在表达式中有时也会被转换为该函数所在内存区域的首地址，类似数组。

定义：returnType(pointerName)(param list);
举例：int (pmax)(int,int)=max; 没有形参用void
函数指针的名字pmax，int表示该函数指针指向的函数是返回int类型的，max函数名。使用(*pmax)(x,y)或者pmax(x,y)调用函数

1.returnType 为函数返回值类型
2.pointerName 为指针名称
3.param list 为函数参数列表
4.参数列表中可以同时给出参数的类型和名称，也可以只给出参数的类型。
5.括号不能省略，如果省略就成了函数原型，表明返回值类型为returnType *(也就是指针函数)

动态内存分配

全局变量 – 内存中的静态存储区
非静态的局部变量 – 内存中的动态存储区 – stack栈
临时使用的数据 – 建立动态内存分配区域，需要时随时开辟，不需要时及时释放 – heap堆

根据需要向系统申请所需大小的空间，由于未在声明部分定义其为变量或者数组，不能通过变量名或者数组名来引用这些数据，只能通过指针

内存动态分配的相关函数
头文件 #include <stdlib.h>声明了四个关于内存动态分配的函数

1.函数原型void *malloc(unsigned int size)
malloc – memory allocation
作用：在内存的动态存储区(heap)中分配一个长度为size的连续空间
说明：形参size的类型为无符号整型，函数返回值是所分配区域的第一个字节的地址，即此函数是一个指针型函数
举例:malloc(100);开辟100字节的临时空间，返回值为其第一个字节的地址

2.函数原型void *calloc(unsigned n,unsigned size)
作用：在内存的动态存储区(heap)分配n个长度为size的连续空间，一般用于数组
说明：函数返回指向所分配域起始位置的指针，分配不成功，返回NULL
举例：用calloc函数可以为一维数组开辟动态存储空间，n为数组元素的个数，每个元素的长度为size

3.函数原型void free(void *p)
作用：释放变量p指向的动态空间，使这部分空间能重新被其他变量使用。
说明：free函数无返回值
举例：free(p)

4.原型函数void realloc(void p,unsigned int size)
作用：重新分配malloc或calloc函数获得的动态空间大小，将p指向的动态空间大小改为size，p值不变，分配失败返回NULL

返回类型说明
void 指针称为无类型指针，不指向哪一种具体的类型数据，只表示用来指向一个抽象类型的数据，即仅提供一个纯地址(不能用 无类型指针获取值)，而不能指向任何具体的对象。

说明1：当把void指针赋值给不同基类型的指针变量时，编译系统会自动进行转换(c99)，不必用户自己进行强制转换
说明2：赋值后p3得到a的纯地址，但并不指向a，不能通过*p3来获取a的值

//举例1
int a= 3;
void *p3 = NULL;
p3 = &3; //p3 =(void *)&a
//举例2
int *p;
p = (int *)malloc(5*sizeof(int));//开辟了5*4=20字节的空间 (int *) 表示void* ->int*
//c99可以如下写法
p = malloc(5*sizeof(int));
for(i=0;i<5;i++){
    scanf(%d,p+1); //这里地址+1个单位,相当于+4个字节
}

动态分配内存的基本原则
1.避免分配大量的小内存块，分配heap上的内存有一些系统开销，所以分配许多小的内存块比分配几个大内存块的系统开销大
2.仅在需要时分配内存并及时释放，否则出现内存泄漏。
3.在循环中分配内存时，要小心
